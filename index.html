❓ Aide — comment utiliser l'app
Un petit guide simple, étape par étape
Bienvenue ! Cette application sert à suivre le classement du Défi Double et à enregistrer les résultats de vos matchs. Pas besoin d'installer quoi que ce soit. Voici à quoi servent les différents onglets 👇
1 Choisir Adulte ou Junior 2 La Calculatrice 3 Les Équipes (classement + profils) 4 La Saisie d'un match 5 Les Matchs (historique) 6 Le circuit Junior 7 En cas de souci

1 · Choisir Adulte ou Junior

Tout en haut de l'écran, un bouton permet de basculer entre les deux tournois : le Défi Double (Adulte) et le Défi Junior. Touchez celui qui vous concerne : les onglets en dessous changent automatiquement.

2 · La Calculatrice

À quoi ça sert : savoir à l'avance combien de points rapporterait un match, sans rien enregistrer. Pratique pour se faire une idée avant de jouer.

  • 1Entrez la cote de votre équipe et celle de l'équipe adverse.
  • 2Choisissez qui gagne (et s'il y a eu un 3ᵉ set).
  • 3L'app affiche les points gagnés par chaque équipe. C'est juste une simulation.

3 · Les Équipes (classement)

À quoi ça sert : voir le classement en direct de toutes les équipes, avec leurs points. Vous pouvez trier par classement, par cote, ou par ordre alphabétique.

💡 Astuce : touchez le nom d'une équipe pour ouvrir son profil. En cliquant, vous voyez toutes les statistiques de l'équipe sur laquelle vous avez cliqué : ses points, ses victoires et défaites, sa série en cours, son meilleur match…

4 · La Saisie d'un match

À quoi ça sert : enregistrer le résultat d'un match. Pour que ce soit sécurisé, il faut le code PIN (4 chiffres) des deux équipes. Ça se fait en 3 étapes :

  • 1Choisissez votre équipe, puis tapez votre code PIN. Un ✓ vert apparaît si c'est bon.
  • 2Choisissez l'équipe adverse, et votre adversaire tape SON code PIN.
  • 3Pour chaque set, entrez le nombre de jeux de chaque équipe. Le vainqueur est calculé automatiquement — rien à cocher. Vérifiez, puis validez : c'est enregistré !
🔑 Le code PIN est le code secret à 4 chiffres que vous avez reçu par message. Chaque équipe a le sien ; il évite que quelqu'un enregistre un résultat à votre place.
💡 La ligne gagnante de chaque set passe en vert et le vainqueur s'affiche tout seul. Pas besoin de saisir les scores « dans le bon sens » : tant qu'un résultat est impossible (set à égalité, ou 1 set partout), le bouton Confirmer reste grisé.

5 · Les Matchs (historique)

À quoi ça sert : revoir tous les matchs déjà joués, du plus récent au plus ancien, avec les scores et les points. Vous pouvez filtrer par équipe pour ne voir que leurs matchs.

6 · Le circuit Junior

Le tournoi Junior fonctionne pareil, avec quelques différences :

  • ★Le classement se fait en étoiles : +2 ⭐ pour le vainqueur, +1 ⭐ pour le perdant.
  • 🟢Certains joueurs jouent avec des balles vertes : l'app vous prévient si c'est le cas.
  • =Les onglets Joueurs, Saisie et Matchs marchent comme côté adulte.

7 · En cas de souci

Si un message « Serveur momentanément indisponible » apparaît : pas de panique, attendez quelques secondes et réessayez. Si ça persiste, prévenez l'organisateur.

Si vous avez oublié votre code PIN, demandez à l'organisateur du tournoi.

Modifier le match
Gagnant = +cote adverse · Perdant = +1/2/3 pts (selon cote) · 3 sets = +1 pt bonus pour le perdant
