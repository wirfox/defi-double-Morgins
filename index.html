Accès administrateur
Première utilisation. Choisis ton code admin (6 à 20 caractères). Le code est hashé (SHA-256) : il n'apparaît jamais dans le code source ni sur GitHub. À toi de bien le retenir.
Initialisation du tournoi
À faire une seule fois en début de saison. Génère les 22 équipes dans Firestore avec un PIN aléatoire chacune.
Ajouter une équipe
Crée une nouvelle équipe dans Firestore (PIN aléatoire chiffré comme les autres). Utile si une équipe rejoint le tournoi en cours de route.

Outil de réparation
Si les compteurs de points sont incohérents (négatifs, etc.), ce bouton les recalcule depuis zéro en additionnant les pointsA et pointsB de chaque match enregistré. Aucun match n'est supprimé.
⚠️ Reset Défi Double (Adulte)
Supprime tous les matchs du tournoi adulte et remet tous les compteurs de points à 0. Les équipes et leurs codes PIN restent inchangés. Le tournoi junior n'est PAS touché.
⚠️ Reset Défi Junior
Supprime tous les matchs juniors et remet tous les compteurs d'étoiles à 0. Les joueurs et leurs codes PIN restent inchangés. Le tournoi adulte n'est PAS touché.
